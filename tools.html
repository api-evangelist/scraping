These are the common tools I have pullled from my research so far.
Apache Tika

The Apache Tikatrade toolkit detects and extracts metadata and structured text content from various documents using existing parser libraries
PhantomJS

PhantomJS is a headless WebKit scriptable with a JavaScript API It has fast and native support for various web standards DOM handling CSS selector JSON Canvas and SVG
Portia

Portia is a tool for visually scraping web sites without any programming knowledge Just annotate web pages with a point and click editor to indicate what data you want to extract and portia will learn how to scrape similar pages from the site Portia has a web based UI served by a Twisted server so you can install it on almost any modern platform
Scraper - Google Chrome Extension

Scraper is a simple data mining extension for Google Chrometrade that is useful for online research when you need to quickly analyze data in spreadsheet form
Scrapy

Scrapy is a fast highlevel screen scraping and web crawling framework used to crawl websites and extract structured data from their pages It can be used for a wide range of purposes from data mining to monitoring and automated testing
