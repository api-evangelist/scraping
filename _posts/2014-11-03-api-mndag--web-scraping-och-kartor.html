Veckans API-nyheter handlar om ett nytt verktyg för att skapa APIer baserat på sajter och en användbar uppdatering av Google Maps API.Ännu mer om APIer och öppna data får ni om ni stalkar @andreaskrohn på Twitter.Det finns väldigt mycket data på webben idag, men det är inte alltid så lätt att komma åt den.Man kan skriva egna script för att skrapa en sajt, eller använda tjänster som diffbot, kimono eller import.io.
