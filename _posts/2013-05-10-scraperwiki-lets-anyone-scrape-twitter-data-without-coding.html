Summary: A new beta version of ScraperWiki makes it easy to relatively easy to scrape Twitter for certain phrases and get to work analyzing the data.It’s just one more way that data analysis is getting democratized.The Obama administration’s open data mandate announced on Thursday was made all the better by the unveiling of the new ScraperWiki service on Friday.If you’re not familiar with ScraperWiki, it’s a web-scraping service that has been around for a while but has primarily focused on users with some coding chops or data journalists willing to pay to have someone scrape data sets for them.Its new service, though, currently in beta, also makes it possible for anyone to scrape Twitter to create a custom data set without having to write a single line of code.
